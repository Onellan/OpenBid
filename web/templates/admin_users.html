Role scope guide

  • Platform Super Admin has full control across OpenBid, including platform users, tenant management, sources, health, and audit operations.
  • Platform Admin can run platform operations across tenants, but cannot take super-admin-only actions.
  • Tenant Owner is the highest authority inside a tenant and can manage tenant settings, admins, and users.
  • Tenant Admin manages tenant users and settings, but cannot grant ownership.
  • Super User can handle advanced day-to-day operational work inside the tenant without administrative ownership powers.
  • User handles normal operational workflows inside the tenant.
  • Viewer has read-only access inside the tenant.
