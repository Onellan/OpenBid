This repository has been cleaned into a true SQLite-only runtime codebase.

Removed:
- internal/store/jsonstore.go
- internal/store/jsonstore_test.go
- cmd/json_to_sqlite

Kept:
- internal/store/sqlitestore.go as the only runtime store implementation
- SQLite-backed app startup
- SQLite default data paths in local and Docker execution

Important:

- Existing JSON data is not migratable by the repo itself anymore because the one-time migration utility was removed in this cleanup pass.
