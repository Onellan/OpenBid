# Installation Instructions

OpenBid installation and setup instructions have been consolidated into the root [README.md](../README.md).

Use the README as the single source of truth for:

- Raspberry Pi prerequisites and OS recommendation
- Docker and Docker Compose installation
- environment variable setup
- build and start commands
- access instructions
- persistent storage and volumes
- logs
- restart, stop, update, backup, and restore
- troubleshooting
- auto-start after reboot

This folder is retained only for legacy deployments that already use its Compose bundle. New installs should start from the repository root and follow [../README.md](../README.md).
